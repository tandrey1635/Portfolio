В 2021 году в Индустриальном парке «Есипово» дан старт реализации проекта по созданию первого в России фармацевтического кластера, одного из крупнейших фармацевтических промышленно-логистических комплексов в Восточной Европе.
На территории совмещены научные разработки, промышленное производство и логистика.
Один из крупнейших в стране инвестиционных проектов, призванный объединить в один фармацевтический кластер научную базу по разработке новых препаратов, производственный комплекс по производству современных лекарств различных форм выпуска и логистический комплекс с возможностью дистрибуции по всей территории страны и стран ближнего зарубежья. ФПЛК «Есипово» станет единым центром инновационных разработок, объединив все исследовательские центры компании Авексима, а новые производственные мощности позволят компании выйти на лидирующие позиции по объему выпускаемых препаратов.
